    Usage Guide

    Learn how to effectively use TaskHub to manage your tasks and collaborate with your team.

    Getting Started with Tasks

    Creating a New Task

    1. Click the "+ New Task" button
    2. Enter a task title
    3. Add a description (optional)
    4. Set a due date and priority level
    5. Assign to team members (optional)
    6. Click "Create Task"

    Task Properties

    • Title: Brief name of the task
    • Description: Detailed information about the task
    • Due Date: When the task should be completed
    • Priority: Urgency level (Low, Medium, High, Critical)
    • Status: Current state (To Do, In Progress, Under Review, Done)
    • Assignee: Team member responsible for the task
    • Tags: Custom labels for categorization

    Project Management

    Create a Project

    1. Click "Projects" in the sidebar
    2. Select "+ New Project"
    3. Enter project name and description
    4. Choose project type (Personal, Team, Public)
    5. Click "Create"

    Organizing Tasks in Projects

    • Drag tasks between projects
    • Use folders within projects for better organization
    • Create project templates for recurring workflows
    • Set project privacy levels

    Task Workflows

    Updating Task Status

    1. Open a task
    2. Click the Status dropdown
    3. Select the new status
    4. Save changes automatically

    Adding Comments

    1. Open a task
    2. Scroll to the Comments section
    3. Type your comment
    4. Press Enter or click "Post"
    5. Mention teammates using @username

    Attaching Files

    1. Open a task
    2. Click "+ Attach" button
    3. Select file from your computer
    4. Or drag and drop files directly

    Filtering and Searching

    Filter Tasks

    • Click "Filter" icon
    • Select filter criteria:
    • Status (To Do, In Progress, etc.)
    • Priority (Low, Medium, High, Critical)
    • Assignee
    • Due date range
    • Tags

    Search Tasks

    1. Click the Search icon
    2. Type keywords in the search box
    3. Results update in real-time
    4. Click a result to open the task

    Save Custom Views

    1. Set your filters
    2. Click "Save View"
    3. Name your view
    4. Access saved views from the sidebar

    Collaboration Features

    Sharing Tasks

    1. Open a task
    2. Click "Share" button
    3. Enter team member email
    4. Select permission level (View, Edit, Comment)
    5. Click "Share"

    Team Comments

    • Add comments to collaborate on tasks
    • Reply to specific comments
    • Mention team members with @
    • Pin important comments

    Activity Timeline

    • View all task changes in the activity log
    • See who made changes and when
    • Track task history

    Notifications

    Setting Preferences

    1. Click your profile icon
    2. Select "Settings"
    3. Go to "Notifications"
    4. Configure preferences:
    5. Choose notification types
    6. Select frequency (Real-time, Daily, Weekly)
    7. Pick channels (Email, In-app, SMS)

    Managing Notifications

    • Mute notifications for specific projects
    • Snooze notifications temporarily
    • Mark notifications as read

    Reports and Analytics

    View Reports

    1. Navigate to "Reports" section
    2. Choose report type:
    3. Task completion rate
    4. Team productivity
    5. Project progress
    6. Time tracking

    Export Data

    1. Open a report
    2. Click "Export"
    3. Choose format (CSV, PDF, Excel)
    4. Download the file

    Keyboard Shortcuts

    Shortcut Action
    Ctrl + N New task
    Ctrl + F Search
    Ctrl + K Command palette
    Ctrl + , Settings
    Escape Close modal
    Tab Navigate fields
    @ Mention user

    Tips and Best Practices

    Effective Task Organization

    • Use clear, descriptive task titles
    • Add relevant tags for easy filtering
    • Set realistic due dates
    • Break large tasks into smaller subtasks

    Team Collaboration

    • Assign tasks clearly
    • Use comments for context
    • Mention relevant team members
    • Review team workload regularly

    Staying Productive

    • Check your dashboard daily
    • Use filters to focus on priorities
    • Archive completed projects
    • Review and adjust your workflows

    Troubleshooting

    Task Not Updating

    • Refresh your browser (Ctrl + R)
    • Check your internet connection
    • Log out and log back in

    Notifications Not Working

    • Verify notification settings in preferences
    • Check browser notification permissions
    • Ensure your email is verified

    Sync Issues

    • Desktop app: Restart the application
    • Mobile: Force close and reopen
    • Web: Clear browser cache
