    Getting Started

    Get up and running with TaskHub in just a few minutes.

    Prerequisites

    Before you begin, make sure you have: - A valid email address - Internet connection - A modern web browser (Chrome, Firefox, Safari, Edge)

    Step 1: Create Your Account

    1. Visit taskhub.app
    2. Click the "Sign Up" button
    3. Enter your email address
    4. Create a strong password (minimum 8 characters)
    5. Click "Create Account"
    6. Check your email for a verification link
    7. Click the link to confirm your email
    8. You're now ready to use TaskHub!

    Step 2: Complete Your Profile

    1. Click your profile icon in the top-right corner
    2. Select "Profile Settings"
    3. Add your full name
    4. Upload a profile picture (optional)
    5. Set your timezone
    6. Choose your preferred language
    7. Click "Save"

    Step 3: Create Your First Project

    1. Click "Projects" in the left sidebar
    2. Click "+ New Project"
    3. Enter a project name (e.g., "My First Project")
    4. Add a description (optional)
    5. Choose project type:
    6. Personal: Only you can access
    7. Team: Share with team members
    8. Public: Anyone with link can view
    9. Click "Create Project"

    Step 4: Add Your First Task

    1. Click your newly created project
    2. Click "+ New Task"
    3. Enter a task title (e.g., "Welcome to TaskHub")
    4. Add a description (optional)
    5. Set a due date (optional)
    6. Set priority: Low, Medium, High, or Critical
    7. Click "Create Task"

    Congratulations! You've created your first task.

    Step 5: Invite Team Members (Optional)

    Adding Team Members

    1. Click "Team" in the sidebar
    2. Click "+ Invite Members"
    3. Enter team member email addresses (comma-separated)
    4. Select their role:
    5. Member: Can create and edit tasks
    6. Admin: Can manage team and settings
    7. Click "Send Invites"

    Team members will receive an invitation email and can join your workspace.

    Essential Features to Know

    Managing Tasks

    Update Task Status: 1. Open a task 2. Click the status dropdown 3. Change to: To Do, In Progress, Under Review, or Done

    Add Comments: 1. Open a task 2. Scroll to Comments section 3. Type your message 4. Mention teammates using @username 5. Click "Post"

    Attach Files: 1. Open a task 2. Click "+ Attach" 3. Select files or drag and drop 4. Files are stored with the task

    Organizing Your Work

    Create Tags: 1. Open a task 2. Click "+ Add Tag" 3. Enter tag name (e.g., "Urgent", "Bug", "Feature") 4. Tags help filter and organize tasks

    Filter Tasks: 1. Click the "Filter" icon 2. Select criteria (Status, Priority, Assignee, Tags) 3. View only relevant tasks

    Search Tasks: 1. Click the Search icon 2. Type keywords 3. Results appear instantly

    Dashboard Overview

    Your TaskHub dashboard shows:

    • Quick Stats: Task completion overview
    • Recent Tasks: Tasks you recently worked on
    • Assigned to Me: Tasks assigned to you
    • Upcoming Deadlines: Tasks due soon
    • Team Activity: Recent changes by team members

    Customizing Your Workspace

    Notification Settings

    1. Click your profile icon
    2. Select "Settings"
    3. Go to "Notifications"
    4. Choose notification types:
    5. Task assignments
    6. Deadline reminders
    7. Comments
    8. Project updates
    9. Select frequency and channels
    10. Click "Save"

    Appearance Preferences

    1. Go to "Settings"
    2. Click "Appearance"
    3. Choose theme: Light, Dark, or Auto
    4. Adjust sidebar width
    5. Enable/disable animations
